The lm() Function

   method = "qr", model = TRUE, x = FALSE, y = FALSE, qr = TRUE,

The lm() (linear model) function is used for fitting linear models. Notice that there are many arguments for this function, but the formula argument is the only argument that needs to be specified. If youâ€™d like to learn more about functions and arguments, please take a look at Chapter 2 which covers basic programming concepts, including functions and arguments.

Letâ€™s take a look at how we can use the lm() function to make two models with the same variables using a simple regression: the untransformed linear model and a mean-centered model. For both models weâ€™ll use Sepal Length as the dependent variable and Sepal Width as the independent variable.

\[y_i\ =\ \beta_0\ +\ \beta_1\left(x_i\right)\ +\ e_i\] The equation above represents the untransformed, simple regression model with one continuous predictor. Letâ€™s implement this model with the lm() function. The dependent variable is listed first, followed by a tilda, ~, and then the independent variable(s).

Model 2: Mean-Centered

\[y_i=\beta_0+\beta_1\left(x_i\ -\ \overline{x}\right)\ +\ e_i\] In this example weâ€™ll use the same variables as before but this time weâ€™ll mean-center the independent/predictor variable. First, we create a column that consists of the sepal width mean, which is 3.06. Since there are 150 rows in the iris dataset, that means we are creating a column that has the value 3.06 repeated 150 times. This value is then saved into the column SW_mean; thatâ€™s what the first line of code in the code chunk below is doing. In the second line of code, each sepal width value is subtracted from the sepal width mean column, which is then stored in a new column called SW_mean_center.

# create a column that consists of the mean sepal width value
iris$SW_mean <- mean(iris$Sepal.Width) 

# subtract sepal width column from mean column
iris$SW_mean_center <- iris$Sepal.Width - iris$SW_mean

Letâ€™s see what it looks like.
